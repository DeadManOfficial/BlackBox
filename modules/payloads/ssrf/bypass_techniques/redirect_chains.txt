# Protocol downgrade
# HTTPS -> HTTP redirect to bypass HTTPS-only filters
# https://your-server.com/ -> http://internal-service/

# DNS rebinding
# 1. Set up DNS with low TTL
# 2. First request resolves to allowed IP
# 3. DNS changed to internal IP before second request
# 4. Second request hits internal service
